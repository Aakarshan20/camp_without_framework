路線一

國道1號苗栗/公館交流道下 → 接【128】(往苗栗方向) → 約1.5公里左轉台72快速道路(往汶水) → 台72線終點右轉經汶水橋，接台3線 →過橋(右前方有7-11超商)立即左轉苗62清安道路 → 清安豆腐街 → 右轉苗62-1道路 → 叉路右轉經天然居 →叉路鷂婆山H901登山口，左轉途經1公里山區產業道路 (沿路均有指標) → 先到鑽石林 → 再往上走即可見到「逸視界」。



路線二

大湖中原路《台3線》→ 大湖酒莊 → 進入大湖市區 → 叉路口左轉中山路 → 左轉民族路 → 大湖國中前右轉苗61綫 → 小南勢沿指標左轉上坡→ 經泰安國中 → 遇警察局走左側道路 → 往前500公尺叉路右轉直上 → 約經2.2公里山區產業道路(沿路均有指標) → 先到鑽石林 → 再往上走即可見到「逸視界」。

【訂位須知】


1.預約時間為(週一至週五)上午9：00至晚上8：00，訂位專線0921-636063， 平日都在營區整理環境，若有漏接電話，敬請見諒，我們會儘速回覆。

2.您也可以在我們的FB粉絲專頁洽詢，https://www.facebook.com/easycamping，我們會儘速回覆您的留言。

3.預訂營位請務必留下手機號碼及全名，以免產生錯誤。經電話預訂確認OK後，請於訂位日起算一週內支付訂金，若於保留時間內未支付訂金，則不保留營位，讓給其他候補之露營朋友，不另行通知。

4.匯款後，請務必來電告知「匯款帳號末4碼」及留下您的「姓名」、「聯絡電話」、「訂位日期」、「人數及車數」，即完成預訂手續。

5.我們於收到訂金後，將為您保留營地，剩餘款項，於到達營區後再行收取。

6.晚上8點前收取剩餘營地費用，請知悉。

7.12月至隔年4月為草莓季，亦為露營旺季，建議提前一至二個月訂位，以免向隅。
